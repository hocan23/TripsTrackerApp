Description

The app is about to track all user location on map while moving. When the user stops for five minutes it ends the trip , the app is working in background.

Features Used:
CoreData
Firebse Event Tracking
MVVM Design Patern
Swift MapKit

Requirements

iOS 13.0 and later
XCode Version 13.1

Open it with XCode
